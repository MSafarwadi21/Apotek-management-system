Tutorial Hosting dengan GitHub

Berikut langkah-langkah hosting menggunakan GitHub Pages:

  1. Buat repository baru di GitHub.
  2. Upload semua file proyek Anda ke repository tersebut.
  3. Pergi ke Settings > Pages.
  4. Pilih branch main dan folder /root.
  5. Klik Save, lalu tunggu beberapa menit.
  6. Situs Anda akan live di https://username.github.io/repository.
